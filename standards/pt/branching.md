# Estratégia de Ramificação

## Convenção de Nomenclatura

```
issue-<id>-<slug>
```

- `<id>`: número da issue do `known_issues.md` ou rastreador remoto
- `<slug>`: descrição em minúsculas com hífens

Exemplos:
- `issue-3-weak-url-validation`
- `issue-12-add-analytics-endpoint`

## Fluxo de Trabalho

1. Crie o branch a partir do branch padrão
2. Implemente as alterações e faça commit
3. Envie e abra um PR/MR
4. Após o merge, exclua o branch
